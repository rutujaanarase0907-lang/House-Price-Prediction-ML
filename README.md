# House Price Prediction ML Model

## Project Overview
This project predicts house prices using Machine Learning algorithms based on different housing features.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis
- Feature Selection
- Model Training
- Model Evaluation
- Price Prediction

## Model Performance
R² Score: 0.918

## Algorithm Used
- Linear Regression

## Project Output
The model predicts house prices and compares actual prices with predicted prices.
